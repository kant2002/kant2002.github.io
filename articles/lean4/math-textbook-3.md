**–ü–æ–∫—Ä–æ–∫–æ–≤–µ –ø–æ—è—Å–Ω–µ–Ω–Ω—è –¥–æ–≤–µ–¥–µ–Ω–Ω—è:**

**–©–æ –º–∏ –¥–æ–≤–æ–¥–∏–º–æ**: —Ç–≤–µ—Ä–¥–∂–µ–Ω–Ω—è $\varnothing \subseteq A$ –æ–∑–Ω–∞—á–∞—î ¬´–ø–æ—Ä–æ–∂–Ω—è –º–Ω–æ–∂–∏–Ω–∞ —î –ø—ñ–¥–º–Ω–æ–∂–∏–Ω–æ—é –¥–æ–≤—ñ–ª—å–Ω–æ—ó –º–Ω–æ–∂–∏–Ω–∏ $A$¬ª. –ó–∞ –Ω–∞—à–∏–º –æ–∑–Ω–∞—á–µ–Ω–Ω—è–º –ø—ñ–¥–º–Ω–æ–∂–∏–Ω–∏ —Ü–µ —Ä–æ–∑–≥–æ—Ä—Ç–∞—î—Ç—å—Å—è –≤ $\forall x,\; x \in \varnothing ‚Üí x \in A$ (–¥–ª—è –≤—Å—ñ—Ö $x$: —è–∫—â–æ $x$ –Ω–∞–ª–µ–∂–∏—Ç—å –ø–æ—Ä–æ–∂–Ω—ñ–π –º–Ω–æ–∂–∏–Ω—ñ, —Ç–æ $x$ –Ω–∞–ª–µ–∂–∏—Ç—å $A$).

**–ö–ª—é—á–æ–≤–µ —Å–ª–æ–≤–æ `by`**: –≤–æ–Ω–æ –ø–µ—Ä–µ–≤–æ–¥–∏—Ç—å —É —Ç–∞–∫—Ç–∏—á–Ω–∏–π —Ä–µ–∂–∏–º, —É —è–∫–æ–º—É –¥–æ–∫–∞–∑ –±—É–¥—É—î—Ç—å—Å—è –ø–æ–∫—Ä–æ–∫–æ–≤–æ –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é —Ç–∞–∫—Ç–∏–∫, –∑–∞–º—ñ—Å—Ç—å —Ç–æ–≥–æ, —â–æ–± –æ–¥—Ä–∞–∑—É –Ω–∞–¥–∞—Ç–∏ –ø–æ–≤–Ω–∏–π —Ç–µ—Ä–º –¥–æ–∫–∞–∑—É.

**–¢–∞–∫—Ç–∏–∫–∞ `intro x h`**: –æ—Å–∫—ñ–ª—å–∫–∏ –Ω–∞—à–∞ —Ü—ñ–ª—å ‚Äî —Ü–µ —Ç–≤–µ—Ä–¥–∂–µ–Ω–Ω—è
$\forall x,\; x ‚àà ‚àÖ \to x ‚àà A$,
–Ω–∞–º –ø–æ—Ç—Ä—ñ–±–Ω–æ –¥–æ–≤–µ—Å—Ç–∏ —É–Ω—ñ–≤–µ—Ä—Å–∞–ª—å–Ω–µ –≤–∏—Å–ª–æ–≤–ª—é–≤–∞–Ω–Ω—è –∑ —ñ–º–ø–ª—ñ–∫–∞—Ü—ñ—î—é. –¢–∞–∫—Ç–∏–∫–∞ `intro` –ø—Ä–∞—Ü—é—î –∑ –æ–±–æ–º–∞ —á–∞—Å—Ç–∏–Ω–∞–º–∏:

* `intro x` –æ–∑–Ω–∞—á–∞—î: ¬´–≤—ñ–∑—å–º–µ–º–æ –¥–æ–≤—ñ–ª—å–Ω–∏–π –µ–ª–µ–º–µ–Ω—Ç $x$¬ª (—Ü–µ –≤—ñ–¥–ø–æ–≤—ñ–¥–∞—î —á–∞—Å—Ç–∏–Ω—ñ $\forall x$),
* `intro h` –æ–∑–Ω–∞—á–∞—î: ¬´–ø—Ä–∏–ø—É—Å—Ç–∏–º–æ –≥—ñ–ø–æ—Ç–µ–∑—É $h : x \in \varnothing$¬ª (—Ü–µ –≤—ñ–¥–ø–æ–≤—ñ–¥–∞—î —á–∞—Å—Ç–∏–Ω—ñ —ñ–º–ø–ª—ñ–∫–∞—Ü—ñ—ó ‚Üí).

–ü—ñ—Å–ª—è –∑–∞—Å—Ç–æ—Å—É–≤–∞–Ω–Ω—è `intro x h` –Ω–∞—à—ñ –ø—Ä–∏–ø—É—â–µ–Ω–Ω—è –≤–∏–≥–ª—è–¥–∞—é—Ç—å —Ç–∞–∫:

* $x : a$ ‚Äî –¥–µ $x$ —î –¥–æ–≤—ñ–ª—å–Ω–∏–º –µ–ª–µ–º–µ–Ω—Ç–æ–º –¥–µ—è–∫–æ–≥–æ —Ç–∏–ø—É $Œ±$,
* $h : x \in \varnothing$ ‚Äî –º–∏ –ø—Ä–∏–ø—É—Å–∫–∞—î–º–æ, —â–æ $x$ –Ω–∞–ª–µ–∂–∏—Ç—å –ø–æ—Ä–æ–∂–Ω—ñ–π –º–Ω–æ–∂–∏–Ω—ñ.

–ù–∞—à–∞ —Ü—ñ–ª—å —Ç–µ–ø–µ—Ä: –¥–æ–≤–µ—Å—Ç–∏, —â–æ $x ‚àà A$ (—â–æ $x$ –Ω–∞–ª–µ–∂–∏—Ç—å –º–Ω–æ–∂–∏–Ω—ñ $A$).

**–°—É–ø–µ—Ä–µ—á–Ω—ñ—Å—Ç—å**: –∞–ª–µ –ø—Ä–∏–ø—É—â–µ–Ω–Ω—è $h : x ‚àà ‚àÖ$ –Ω–∞—Å–ø—Ä–∞–≤–¥—ñ –æ–∑–Ω–∞—á–∞—î $h : \text{False}$ (–∞–¥–∂–µ –Ω–∞–ª–µ–∂–Ω—ñ—Å—Ç—å –¥–æ –ø–æ—Ä–æ–∂–Ω—å–æ—ó –º–Ω–æ–∂–∏–Ω–∏ –∑–∞–≤–∂–¥–∏ —Ö–∏–±–Ω–∞). –û—Ç–∂–µ, –º–∏ –º–∞—î–º–æ –¥–æ–∫–∞–∑ —Ö–∏–±–Ω–æ—Å—Ç—ñ!

**–¢–∞–∫—Ç–∏–∫–∞ `exact False.elim h`**: —ñ–∑ –¥–æ–∫–∞–∑—É $\text{False}$ –º–æ–∂–Ω–∞ –¥–æ–≤–µ—Å—Ç–∏ –±—É–¥—å-—â–æ. `False.elim` —Ä–µ–∞–ª—ñ–∑—É—î –ø—Ä–∏–Ω—Ü–∏–ø –≤–∏–±—É—Ö—É (ex falso quodlibet) ‚Äî —è–∫—â–æ –≤–∏–Ω–∏–∫–ª–∞ —Å—É–ø–µ—Ä–µ—á–Ω—ñ—Å—Ç—å, —Ç–æ –±—É–¥—å-—è–∫–µ —Ç–≤–µ—Ä–¥–∂–µ–Ω–Ω—è —Å—Ç–∞—î –¥–æ–≤—ñ–¥–Ω–∏–º. –û—Å–∫—ñ–ª—å–∫–∏ $h : \text{False}$, –º–∏ –º–æ–∂–µ–º–æ –∑–∞—Å—Ç–æ—Å—É–≤–∞—Ç–∏ `False.elim h`, —â–æ–± –¥–æ–≤–µ—Å—Ç–∏ –Ω–∞—à—É —Ü—ñ–ª—å $x ‚àà A$.

## –í–ª–∞—Å—Ç–∏–≤—ñ—Å—Ç—å (9): –ö–æ–∂–Ω–∞ –º–Ω–æ–∂–∏–Ω–∞ —î –ø—ñ–¥–º–Ω–æ–∂–∏–Ω–æ—é —Å–∞–º–æ—ó —Å–µ–±–µ (—Ä–µ—Ñ–ª–µ–∫—Å–∏–≤–Ω—ñ—Å—Ç—å)

–¶–µ–π –≤–∏–ø–∞–¥–æ–∫ –≤–∏–≥–ª—è–¥–∞—î –∑–Ω–∞—á–Ω–æ –ø—Ä–æ—Å—Ç—ñ—à–∏–º: –º–∏ –ø—Ä–∏–ø—É—Å–∫–∞—î–º–æ $x \in A$ —ñ –º–∞—î–º–æ –¥–æ–≤–µ—Å—Ç–∏ $x \in A$ ‚Äî —É—Å–µ –ø–æ—Ç—Ä—ñ–±–Ω–µ –≤–∂–µ –≤ –Ω–∞—Å —î!

## –í–ª–∞—Å—Ç–∏–≤—ñ—Å—Ç—å (10): –¢—Ä–∞–Ω–∑–∏—Ç–∏–≤–Ω—ñ—Å—Ç—å –≤—ñ–¥–Ω–æ—à–µ–Ω–Ω—è –ø—ñ–¥–º–Ω–æ–∂–∏–Ω–∏

–†–∞—Å—å–æ–≤–∞ –ø–æ—è—Å–Ω—é—î: ¬´–§–æ—Ä–º—É–ª–∞ (10) —î –∑–∞–∫–æ–Ω–æ–º —Ç—Ä–∞–Ω–∑–∏—Ç–∏–≤–Ω–æ—Å—Ç—ñ –¥–ª—è –≤—ñ–¥–Ω–æ—à–µ–Ω–Ω—è –≤–∫–ª—é—á–µ–Ω–Ω—è. –©–æ–± –¥–æ–≤–µ—Å—Ç–∏ —Ü–µ, –ø—Ä–∏–ø—É—Å—Ç–∏–º–æ, —â–æ $A ‚äÜ B$ —ñ $B ‚äÜ C$. –¢–æ–¥—ñ –∫–æ–∂–µ–Ω –µ–ª–µ–º–µ–Ω—Ç –º–Ω–æ–∂–∏–Ω–∏ $A$ —î –µ–ª–µ–º–µ–Ω—Ç–æ–º –º–Ω–æ–∂–∏–Ω–∏ $B$, –∞ –∫–æ–∂–µ–Ω –µ–ª–µ–º–µ–Ω—Ç –º–Ω–æ–∂–∏–Ω–∏ $B$ —î –µ–ª–µ–º–µ–Ω—Ç–æ–º –º–Ω–æ–∂–∏–Ω–∏ $C$. –ó–≤—ñ–¥—Å–∏ –≤–∏–ø–ª–∏–≤–∞—î, —â–æ –∫–æ–∂–µ–Ω –µ–ª–µ–º–µ–Ω—Ç –º–Ω–æ–∂–∏–Ω–∏ $A$ —î –µ–ª–µ–º–µ–Ω—Ç–æ–º –º–Ω–æ–∂–∏–Ω–∏ $C$, –∞ –æ—Ç–∂–µ $A ‚äÜ C$¬ª.

```lean
theorem subset_trans (A B C : RasiowaSet Œ±) :
  A ‚äÜ B ‚Üí B ‚äÜ C ‚Üí A ‚äÜ C := by
  intro h‚ÇÅ h‚ÇÇ  -- –ü—Ä–∏–ø—É—Å—Ç–∏–º–æ h‚ÇÅ : A ‚äÜ B —Ç–∞ h‚ÇÇ : B ‚äÜ C
               -- –ú–µ—Ç–∞ —Å—Ç–∞—î: A ‚äÜ C (i.e., ‚àÄ x, x ‚àà A ‚Üí x ‚àà C)
  intro x h    -- –ù–µ—Ö–∞–π x –¥–æ–≤—ñ–ª—å–Ω–µ, –ø—Ä–∏–ø—É—Å—Ç–∏–º–æ, —â–æ h : x ‚àà A
               -- –ú–µ—Ç–∞ —Å—Ç–∞—î: x ‚àà C
  have h_B : x ‚àà B := h‚ÇÅ x h  -- –ó–∞—Å—Ç–æ—Å—É—î–º–æ h‚ÇÅ —â–æ–± –æ—Ç—Ä–∏–º–∞—Ç–∏ x ‚àà B —ñ–∑ x ‚àà A
  exact h‚ÇÇ x h_B              -- –ó–∞—Å—Ç–æ—Å—É—î–º–æ h‚ÇÇ —â–æ–± –æ—Ç—Ä–∏–º–∞—Ç–∏ x ‚àà C —ñ–∑ x ‚àà B
```

–ú–∏ –≤–∏–±—É–¥–æ–≤—É—î–º–æ —ñ–º–ø–ª—ñ–∫–∞—Ü—ñ—ó –∫—Ä–æ–∫ –∑–∞ –∫—Ä–æ–∫–æ–º: —Å–ø–æ—á–∞—Ç–∫—É –æ—Ç—Ä–∏–º—É—î–º–æ
`h_B : x ‚àà B` –∑ `h : x ‚àà A`, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ `h‚ÇÅ`, –∞ –ø–æ—Ç—ñ–º –∑–∞—Å—Ç–æ—Å–æ–≤—É—î–º–æ `h_B` —Ä–∞–∑–æ–º —ñ–∑ `h‚ÇÇ`, —â–æ–± –æ—Ç—Ä–∏–º–∞—Ç–∏ `x ‚àà C`.

**–í–ª–∞—Å—Ç–∏–≤—ñ—Å—Ç—å (11): –ê–Ω—Ç–∏—Å–∏–º–µ—Ç—Ä–∏—á–Ω—ñ—Å—Ç—å ‚Äî –≤–∑–∞—î–º–Ω–µ –≤–∫–ª—é—á–µ–Ω–Ω—è –ø—ñ–¥–º–Ω–æ–∂–∏–Ω –æ–∑–Ω–∞—á–∞—î —Ä—ñ–≤–Ω—ñ—Å—Ç—å**

–†–∞—Å—ñ–æ–≤–∞ –ø–æ—è—Å–Ω—é—î: *¬´–©–æ–± –¥–æ–≤–µ—Å—Ç–∏ (11), –ø—Ä–∏–ø—É—Å—Ç—ñ–º–æ, —â–æ $A \subseteq B$ —ñ $B \subseteq A$. –û—Ç–∂–µ, –∫–æ–∂–µ–Ω –µ–ª–µ–º–µ–Ω—Ç –º–Ω–æ–∂–∏–Ω–∏ $A$ —î –µ–ª–µ–º–µ–Ω—Ç–æ–º –º–Ω–æ–∂–∏–Ω–∏ $B$, —ñ –∫–æ–∂–µ–Ω –µ–ª–µ–º–µ–Ω—Ç –º–Ω–æ–∂–∏–Ω–∏ $B$ —î –µ–ª–µ–º–µ–Ω—Ç–æ–º –º–Ω–æ–∂–∏–Ω–∏ $A$. –¢–∞–∫–∏–º —á–∏–Ω–æ–º, –º–Ω–æ–∂–∏–Ω–∏ $A$ —ñ $B$ –º–∞—é—Ç—å –æ–¥–Ω–∞–∫–æ–≤—ñ –µ–ª–µ–º–µ–Ω—Ç–∏, —Ç–æ–±—Ç–æ –≤–æ–Ω–∏ —Ç–æ—Ç–æ–∂–Ω—ñ¬ª.*

**–í–ª–∞—Å—Ç–∏–≤—ñ—Å—Ç—å (12): –ö–æ–Ω—Ç—Ä–∞–ø–æ–∑–∏—Ü—ñ—è –∞–Ω—Ç–∏—Å–∏–º–µ—Ç—Ä–∏—á–Ω–æ—Å—Ç—ñ**

–†–∞—à—ñ–æ–≤–∞ –ø–æ—è—Å–Ω—é—î: ¬´–§–æ—Ä–º—É–ª–∞ (12) –≤–∏–ø–ª–∏–≤–∞—î –∑ —Ñ–æ—Ä–º—É–ª–∏ (11). –Ø–∫–±–∏ –≤–∏–∫–æ–Ω—É–≤–∞–ª–æ—Å—è $A \subseteq B$ —ñ $B \subseteq A$, —Ç–æ –∑–∞ (11) –º–Ω–æ–∂–∏–Ω–∏ $A$ —ñ $B$ –±—É–ª–∏ –± –æ–¥–Ω–∞–∫–æ–≤–∏–º–∏, —â–æ —Å—É–ø–µ—Ä–µ—á–∏—Ç—å –ø—Ä–∏–ø—É—â–µ–Ω–Ω—é $A \neq B$¬ª.

**–ù–æ–≤—ñ —Ç–∞–∫—Ç–∏–∫–∏**: –º–∏ –≤–≤–æ–¥–∏–º–æ `by_cases` –¥–ª—è –∫–ª–∞—Å–∏—á–Ω–æ–≥–æ —Ä–æ–∑–≥–ª—è–¥—É –≤–∏–ø–∞–¥–∫—ñ–≤ ‚Äî –≤–æ–Ω–∞ —Ä–æ–∑–¥—ñ–ª—è—î –¥–æ–≤–µ–¥–µ–Ω–Ω—è –∑–∞–ª–µ–∂–Ω–æ –≤—ñ–¥ —Ç–æ–≥–æ, —ñ—Å—Ç–∏–Ω–Ω–µ —á–∏ —Ö–∏–±–Ω–µ —Ç–≤–µ—Ä–¥–∂–µ–Ω–Ω—è. –ú–∏ —Ç–∞–∫–æ–∂ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î–º–æ —Ç–∞–∫—Ç–∏–∫–∏ `left` —Ç–∞ `right`: –∫–æ–ª–∏ —Ü—ñ–ª–ª—é —î –¥–∏–∑‚Äô—é–Ω–∫—Ü—ñ—è $(P ‚à® Q)$, –∑–∞—Å—Ç–æ—Å–æ–≤—É—î–º–æ `left`, —â–æ–± –¥–æ–≤–µ—Å—Ç–∏ $P$, –∞–±–æ `right`, —â–æ–± –¥–æ–≤–µ—Å—Ç–∏ $Q$.

```lean
    intro h‚ÇÇ  -- –ü—Ä–∏–ø—É—Å—Ç–∏–º–æ h‚ÇÇ : B ‚äÜ A (–¥–ª—è –ø—Ä–æ—Ç–∏—Ä—ñ—á—á—è)
              -- –ú–µ—Ç–∞ —Å—Ç–∞—î: False

```
